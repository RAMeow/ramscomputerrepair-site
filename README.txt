RAM'S COMPUTER REPAIR - Cloudflare Pages package

Deploy:
1. Unzip this package.
2. In Cloudflare Dashboard, open Workers & Pages.
3. Create a new Pages project.
4. Choose Upload assets.
5. Upload the CONTENTS of the site folder.
6. Add your custom domains:
   - ramscomputerrepair.net
   - www.ramscomputerrepair.net

Notes:
- Static site. No build step required.
- Includes Google Maps embed for Harlingen.
- Includes Google Reviews QR code.
- Uses uploaded logo and review QR assets.
